# RAE-core (Reflective Agentic Engine Core)

The cognitive substrate and communication protocol for the Silicon Oracle RAE Suite v3.

## Overview
RAE-core defines the "DNA" of the agentic factory. It provides versioned Pydantic v2 models for:
- **Evidence Trail**: ActionRecord, DecisionEvidenceRecord, OutcomeRecord.
- **Behavioral Governance**: BehaviorGuarantee, BehaviorSignal, BehaviorViolation.
- **Economic Model**: CostVector and NCU (Normalized Compute Unit) schemas.
- **Evolution**: Hypothesis, InsightPack, FailurePatternPack.
- **Fabric**: MessageEnvelope and Capability schemas.

## Role in the Ecosystem
- **Universal Protocol**: The common language between Phoenix, Hive, Lab, and Quality.
- **Validation Layer**: Enforces semantic correctness across all memory records.
- **Economic Standard**: Standardizes resource consumption tracking.
